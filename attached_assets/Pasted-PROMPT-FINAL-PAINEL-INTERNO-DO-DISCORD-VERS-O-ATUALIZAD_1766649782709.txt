FLUXO — CONFIGURAR CARGOS

1. Usuário autorizado clica em "Configurar Cargos"
2. Bot envia embed com cargos atuais configurados
3. Botões exibidos:
   - Definir Cargo de Moderador
   - Definir Cargo de Administrador

--------------------------------------------------

FLUXO — DEFINIR UM CARGO

1. Staff clica em um botão (ex: Definir Cargo de Moderador)
2. Bot pergunta no chat:
   "Mencione o cargo ou envie o ID do cargo."
3. Staff envia a menção ou ID
4. Bot envia UMA embed contendo apenas descrição:

   "O cargo escolhido foi @CargoSelecionado
    Deseja prosseguir?"

5. Abaixo da embed, três botões:
   - CONFIRMAR
   - EDITAR
   - CANCELAR

--------------------------------------------------

COMPORTAMENTO DOS BOTÕES

CONFIRMAR
- Salva o cargo
- Atualiza as permissões automaticamente
- Retorna para a tela anterior de Configurar Cargos

EDITAR
- Bot pergunta novamente no chat:
  "Mencione o cargo ou envie o ID do cargo."

CANCELAR
- NÃO cancela o !painel
- Apenas retorna para a última página aberta
- Nenhuma configuração é salva
